# 生命科学概論A 知識確認クイズ

早稲田大学「生命科学概論A（化学・応化）」の過去問127問（2009・2010・2017・2018・2019・2021年度）を演習できるクイズアプリです。

## 使い方（そのまま開く場合）

`index.html` をダブルクリックしてブラウザで開くだけで動きます。

## GitHub Pagesで公開する方法

友達にURLひとつで共有したい場合は、GitHub Pagesを使うと無料で公開できます。

1. GitHubで新しいリポジトリを作成する（例: `biology-quiz`）。Public（公開）にしてください。
2. このフォルダの `index.html` をリポジトリにアップロードする。
   - GitHubのウェブ画面から「Add file」→「Upload files」で `index.html` をドラッグ&ドロップするだけでOKです。
3. リポジトリの「Settings」→左メニューの「Pages」を開く。
4. 「Build and deployment」の「Source」を `Deploy from a branch` にし、Branchを `main`（または `master`）、フォルダを `/ (root)` にして「Save」。
5. 数分待つと、ページ上部に公開URLが表示されます（例: `https://ユーザー名.github.io/biology-quiz/`）。
6. そのURLを友達に共有すれば、誰でもブラウザからアクセスできます。

## 記録について

- 各問題の正答・誤答の回数は、**そのブラウザ（そのデバイス）のlocalStorageだけ**に保存されます。
- 友達と記録が共有されることはなく、各自のスマホ・PCごとに別々に記録が溜まります。
- 「成績・苦手問題を見る」画面で、自分がよく間違える問題を一覧できます。
- 「記録をリセット」ボタンで、そのデバイスの記録だけを消せます。

## ファイル構成

- `index.html` — アプリ本体（これ1ファイルだけで完結。サーバー不要）

## 更新したい場合

問題データは `index.html` 内の `<script id="questions-data" type="application/json">` タグの中にJSON形式でそのまま書かれています。ここを編集すれば問題を追加・修正できます。
